# Financial Performance Analysis – GSK Plc

## Overview
This project analyses the financial performance of GSK Plc over the period 2019–2022 using
financial ratio analysis techniques. The objective is to assess liquidity, profitability, efficiency,
and investment performance to support data-driven financial decision-making.

## Key Analysis Areas
- Liquidity analysis (Current ratio, Quick ratio, Gearing ratio)
- Profitability analysis (Gross margin, Net margin, ROA)
- Efficiency analysis (Inventory, Receivables, Payables)
- Investment analysis (EPS, Dividend per share)

## Tools Used
- Microsoft Excel (data preparation and ratio calculations)
- Financial statements and annual reports
- Analytical reporting and interpretation

## Files in This Repository
- GSK_Financial_Performance_Report.pdf – Detailed financial analysis and conclusions
- GSK_Financial_Data_Ratio_Analysis.xlsx – Supporting calculations and data

## Key Insights
- GSK shows strong profitability despite liquidity constraints
- 2022 reflects improved efficiency post consumer healthcare demerger
- Financial structure indicates controlled deleveraging over time
